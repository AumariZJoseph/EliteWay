# EliteWay – Rentals & Transfers (Demo)

Single-page marketing site for **EliteWay**, a Saint Lucia rentals and transfers company. Front-end only — no backend or live booking.

Place your brand file **`logo.jpg`** in the project root (same folder as `index.html`). It is used in the navbar, footer, form success state, and favicon.

## Run locally

Open `index.html` in a browser, or serve the folder:

```bash
npx serve .
```

## Structure

- `index.html` — all sections (hero, transfers, premium, rentals, fleet, requirements, about, inquiry, footer)
- `css/styles.css` — layout, Caribbean-professional theme (navy, teal, gold)
- `js/main.js` — fleet grid, lightbox, gallery slider, mobile nav, quote form demo
- `assets/terms-placeholder.pdf` — placeholder Terms link

## Features

- Responsive navbar with mobile menu
- Hero with gradient overlay and CTAs
- Premium image rotator (5s auto-advance)
- Fleet cards → lightbox with specifications only
- Inquiry form with service-dependent return date; thank-you state on submit
- Placeholder images via Unsplash (requires network)

## Note

This is a **mockup** for lead generation. Form submissions are simulated in the browser only.
